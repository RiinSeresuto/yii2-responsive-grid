# Responsive Table

✨Smol✨ JavaScript Package to make HTML Tables responsive.

## Requirements

-   Node
-   npm or pnpm

## Instalation

Install Node Modules

```bash
npm install

# or

pnpm install
```

Run hot reload:

```bash
npm run dev

#or

pnpm dev
```

Compile:

```bash
npm run build

#or

pnpm build
```
